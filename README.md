### Основы Python

Хендбук по Python поможет овладеть основным синтаксисом и принципами языка. Для этого не потребуется специальной подготовки — достаточно знаний по информатике, логике и математике на уровне школьной программы. Кроме основных конструкций в учебнике рассмотрены разные подходы к программированию, реализованные на Python. А в последней главе вы прикоснётесь к главной суперсиле языка — большому количеству прикладных библиотек.

### Оглавление

1. [Введение](#введение)
2. [Основы Python](handbook-2-constructions.ipynb)  
    Включает базовые конструкции языка Python: ввод/вывод данных, операции с числами и строками, форматирование, условные операторы и циклы.
3. [Коллекции и работа с памятью](handbook-3-collections.ipynb)  
    Описание строк, списков, кортежей, множества, словарей, а также встроенные инструменты для работы с коллекциями и файловыми данными.
4. [Функции в Python](handbook-4-functions.ipynb)  
    Разбор областей видимости, передачи параметров, лямбда-функций, декораторов, рекурсии и генераторов.
5. [Объектно-ориентированное программирование](handbook-5-oop.ipynb)  
    Покрытие тем, связанных с классами, методами, наследованием и волшебными методами Python.
6. [Библиотеки для работы с данными](handbook-6-libraries.ipynb)  
    Обзор популярных библиотек, таких как `math`, `numpy`, `pandas` и `requests`.
7. [Приложение](#приложение)
